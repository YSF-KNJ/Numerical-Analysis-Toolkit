Instructions

Function Definition Domain
x^n Renvoie la variable x élevée à la puissance n. x ∈ ℝ
a*x^n Renvoie la variable x élevée à la puissance n, multipliée par a. (Polynôme) x ∈ ℝ
log(x) Renvoie le logarithme naturel de x. x ∈ ℝ+*
log(x, base) Renvoie le logarithme de x avec la base spécifiée. x ∈ ℝ+*
exp(x) Renvoie la valeur exponentielle de x, qui est e élevé à la puissance de x. x ∈ ℝ
cos(x) Renvoie le cosinus de x en radians. x ∈ ℝ
sin(x) Renvoie le sinus de x en radians. x ∈ ℝ
tan(x) Renvoie la tangente de x en radians. x ∈ ℝ, sauf pour les valeurs de x pour lesquelles cos(x) = 0.
sqrt(x) Renvoie la racine carrée de x. x ∈ ℝ+
cbrt(x) Renvoie la racine cubique de x. x ∈ ℝ
acos(x) Renvoie le cosinus inverse de x en radians. x ∈ [-1, 1]
asin(x) Renvoie le sinus inverse de x en radians. x ∈ [-1, 1]
atan(x) Renvoie la tangente inverse de x en radians. x ∈ ℝ
cosh(x) Renvoie le cosinus hyperbolique de x. x ∈ ℝ
sinh(x) Renvoie le sinus hyperbolique de x. x ∈ ℝ
tanh(x) Renvoie la tangente hyperbolique de x. x ∈ ℝ
